💻 Fiche d’utilisation du Terminal – Mac (Puce M3)

1 Ouvrir le Terminal

  • Méthode 1 : Finder → Applications → Utilitaires → Terminal
  • Méthode 2 : ⌘ + Espace → taper Terminal → Entrée

Le Terminal permet d’interagir avec ton Mac via des commandes texte.

2 Structure d’une commande

commande [options] [arguments]

Exemple :

ls -l /Users
  • ls → liste les fichiers
  • -l → affiche en liste détaillée
  • /Users → indique le dossier à lister

5 Navigation rapide

  • cd .. → remonter d’un dossier
  • cd ~ → aller au dossier personnel
  • cd / → aller à la racine du système
  • Tab → compléter automatiquement un nom

7 Mises à jour et paquets (Homebrew)

Sur Mac M3, installe Homebrew, un gestionnaire de paquets.

Installation :

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Exemples d’usage :

brew install python
brew install git
brew update && brew upgrade

9 Personnalisation (zsh)

Le Terminal macOS utilise zsh par défaut. Personnalise l’invite et tes alias dans ~/.zshrc.

# Ajouter une couleur à l'invite de commande
export PS1="%F{green}%n@%m %F{blue}%~ %F{yellow}%# %f"

# Ajouter des alias pratiques
alias ll='ls -lAh'
alias cls='clear'
alias dev='cd ~/Documents/dev'

Recharge le fichier :

source ~/.zshrc

14 Astuces pratiques

Tape man [commande] pour le manuel (ex : man ls).
Chaîne plusieurs commandes avec && : mkdir test && cd test && touch fichier.txt
Redirige la sortie : ls > liste.txt (écrase) ou ls >> liste.txt (ajoute).
Exécute en arrière‑plan : commande &
