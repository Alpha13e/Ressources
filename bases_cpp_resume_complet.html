Bases du C++ — Résumé complet

Résumé synthétique des notions principales du langage C++.

1) Structure d’un programme C++

#include <iostream>
using namespace std;

class MaClass { };

int main() { return 0; }
  • #include <iostream> : inclut les E/S standard.
  • using namespace std; : permet d’utiliser cout, cin sans préfixe.

2) Entrées / Sorties

long a;
cout << "Saisir a : ";
cin >> a;
cout << a << " au carré = " << a*a << endl;

cin, cout, cerr sont les flux standards d’E/S.

3) Variables et portée

for (int i = 0; i < n; i++) {
  double val;
}

Les variables existent uniquement dans leur bloc de définition.

4) Références & passage de paramètres

int x = 5;
int &r = x;

void swap(int &a, int &b) {
  int tmp = a; a = b; b = tmp;
}

Les références sont des alias de variables et permettent le passage par référence.

5) Arguments par défaut

void f(int a, int b = 0, int c = 1);
  • Les valeurs par défaut doivent être à la fin.
  • Seulement dans le prototype.

6) Surcharge de fonctions

void g(int,int);
void g(double,int);

Le compilateur différencie selon la signature (types des paramètres).

7) Fonctions inline

inline long carre(long x){ return x*x; }

Remplace l’appel par le code pour gagner du temps d’exécution.

8) Allocation dynamique

double *p = new double;
delete p;

int *t = new int[10];
delete[] t;

9) Classes

class Complexe {
public:
  double re, im;
  void afficher(){ cout << re << " + i" << im << endl; }
};

10) Constructeurs

class Complexe {
public:
  double re, im;
  Complexe(){ re=0; im=0; }
  Complexe(double r,double i){ re=r; im=i; }
};

11) Destructeurs

class Signal {
  double *data;
public:
  Signal(int n){ data = new double[n]; }
  ~Signal(){ delete[] data; }
};

12) Encapsulation

class Complexe {
private:
  double re, im;
public:
  double getRe(){return re;}
  void setRe(double r){re=r;}
};

13) Surcharge d’opérateurs

Complexe operator+(const Complexe &a, const Complexe &b){
  return Complexe(a.getRe()+b.getRe(), a.getIm()+b.getIm());
}

bool operator==(const Complexe &a, const Complexe &b){
  return a.getRe()==b.getRe() && a.getIm()==b.getIm();
}
